# AiraOS

AiraOS es un sistema operativo desarrollado por Axioma Aira donde la inteligencia artificial local es el núcleo del sistema.

El objetivo de AiraOS es que la IA no sea solo una aplicación externa o un chatbot en la nube, sino una parte real del sistema: local, privada, útil y orientada a tareas concretas.

## Qué es AiraOS

AiraOS trabaja con asistentes especializados llamados Litas.

Cada Lita está pensada para ayudar en un ámbito concreto, usando IA local, voz, contexto y herramientas adaptadas a cada necesidad.

AiraOS se basa en tres ideas principales:

- IA local
- Privacidad por diseño
- Asistentes especializados para tareas reales

## Objetivos de AiraOS

- IA local
- Privacidad por diseño
- No vender datos del usuario
- Interacción total por voz
- Asistentes especializados para tareas reales
- Funcionamiento orientado a dispositivos y casos de uso concretos
- Un ecosistema de Litas, no una única aplicación de IA
- Dispositivos especializados en fase de prototipo

## Estado del proyecto
